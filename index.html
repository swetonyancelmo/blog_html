A história da Web

A história da web começou em 1989, quando Tim Berners-Lee, cientista da computação britânico, propôs um sistema para compartilhar informações entre pesquisadores no CERN. Em 1991, a World Wide Web foi lançada, revolucionando a forma como acessamos e compartilhamos informações. Com o tempo, a web evoluiu de páginas estáticas (Web 1.0) para um ambiente mais dinâmico e interativo (Web 2.0), caracterizado por redes sociais, blogs e plataformas de colaboração. Atualmente, estamos na transição para a Web 3.0, que integra tecnologias como inteligência artificial, blockchain e a Internet das Coisas, tornando a web mais descentralizada e orientada por dados.

"The Web is more a social creation than a technical one. I designed it for a social effect—to help people work together—and not as a technical toy." Tim Berners-Lee

Clients e Servers

Clients e servers são componentes fundamentais em redes de computadores. Um client é um dispositivo ou programa que solicita serviços ou informações, como um navegador acessando um site. Já o server é um sistema que processa essas solicitações, armazenando e fornecendo dados ou recursos, como um servidor web hospedando páginas. Essa interação é base para a comunicação na internet e em redes locais.

Server-side e Client-side

Server-side e client-side são termos que explicam onde as coisas acontecem em um sistema web. No server-side, as ações são realizadas no servidor, como processar dados ou acessar um banco de informações antes de enviar o resultado para o usuário. Já no client-side, tudo acontece no dispositivo do usuário, como exibir páginas, interagir com botões ou executar scripts no navegador. Juntos, eles trabalham para criar experiências rápidas e dinâmicas na internet.

Ele suporta várias linguagens de programação, oferece integração com extensões e ferramentas, e possui recursos como depuração, terminal integrado e inteligência de código (IntelliSense), tornando-o ideal para desenvolvedores de todos os níveis.

Estrutura Básica do HTML

A estrutura básica do HTML (HyperText Markup Language) é organizada de forma hierárquica, começando com a declaração (<)!DOCTYPE html(>) para indicar a versão do HTML. Em seguida, o elemento raiz (<)html(>) envolve dois principais blocos: (<)head(>), que contém metadados, como título e links de estilo, e (<)body(>), que inclui o conteúdo visível da página, como textos, imagens e links. Essa estrutura é essencial para criar páginas da web.

Tags

As tags são os elementos básicos do HTML, usadas para estruturar e definir o conteúdo de uma página. Elas são escritas entre colchetes angulares, como (<)p(>) para parágrafos ou (<)h1(>) para títulos. Existem tags de abertura e fechamento, como (<)div(>)(, e tags autossuficientes, como (<)img (/>).()

Atributos de Tags

Os atributos de tags fornecem informações adicionais sobre o elemento. Eles são escritos dentro da tag de abertura e têm o formato nome="valor". Por exemplo, em (<)img src="imagem.jpg" alt="Descrição"(>), os atributos src e alt especificam o caminho da imagem e uma descrição alternativa, respectivamente. Atributos ajudam a personalizar e controlar o comportamento das tags.

Textos, Listas não ordenas e Links

As tags de texto no HTML são usadas para formatar e estruturar conteúdo textual. Por exemplo, (<)p> define parágrafos, (<)strong> deixa o texto em negrito, e (<)em> o destaca em itálico. As listas não ordenadas são criadas com a tag (<)ul>, onde cada item é representado pela tag (<)li>. Elas exibem os itens com marcadores (como pontos ou círculos), sendo úteis para agrupar informações sem uma sequência específica. Os links são criados com a tag (<)a> e conectam a página a outros conteúdos ou sites. O atributo href é usado para especificar o destino, como (<)a href="https://exemplo.com">Clique aqui. Essas tags tornam as páginas mais interativas e navegáveis.
